# SYS-crawlers-block-on-apache2

## Qu 'il fallait savoir sur "crawler".

### Dois-je bloquer les crawlers (les robots d'exploration) ?
Pensez à un robot d'exploration du Web comme un bibliothécaire ou un organisateur qui répare une bibliothèque désorganisée, en rassemblant des catalogues de cartes pour que les visiteurs puissent facilement et rapidement trouver des informations. Cependant, si vous ne voulez pas que les robots explorent et indexent toutes vos pages Web, vous devez les bloquer.
